# AI Engineering Projects

Welcome to the **AI Engineering Projects** repository. This is a collection of various Artificial Intelligence and Machine Learning applications, focusing primarily on Generative AI, Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and Multi-Agent workflows. 

Most web applications here are built using **Flask** for the backend and leverage **Groq** for ultra-fast LLM inference.

---

## 🚀 Repository Overview

This repository contains the following standalone AI projects:

| Project Folder | Description | Tech Stack |
| :--- | :--- | :--- |
| **`AI_Chat_Assistant_Flask_Groq`** | A conversational AI application built with Flask and integrated with Groq API, engineered for seamless, real-time, low-latency streaming responses. | Python, Flask, Groq API, HTML/CSS/JS |
| **`AI_Content_Generator_Flask_Groq`** | A web-based marketing automation tool utilizing Groq API to instantly create optimized blog posts, social media copies, and creative content. | Python, Flask, Groq API |
| **`AI_Customer_Support_Assistant`** | An AI automated customer support assistant which instantly handles technical, billing, sales queries round the clock. | Python, LangGraph, Flask |
| **`AI_Email_Generator_Flask_Groq`** | An automated tool that leverages Groq's LLM inference to dynamically generate professionally tailored emails based on user prompts. | Python, Flask, Groq API |
| **`AI_Multi_Agent_Debate_System`** | A multi-agent AI system, that enables role-specific AI agents to take opposing sides on a user-provided topic and generate structured arguments. | Python, LangGraph, Flask |
| **`AI_PDF_RAG_Chatbot_Flask_Groq_LC_Chromadb`**| A RAG pipeline built with LangChain and ChromaDB to perform semantic search and conversational Q&A over unstructured PDF documents. | Flask, Groq, LangChain, ChromaDB |
| **`AI_SQL_Assistant_Flask_Groq`** | A web-based Text-to-SQL assistant tool that allows users to chat directly with their local relational databases using plain english language. | Flask, Groq, SQLite, Pandas |
| **`Multi-Agent_AI_System`** | A Multi-Agent AI System where specialized AI agents collaborate to break down complex tasks into specialized sub-tasks to deliver high-quality results. | Python, Flask, Groq |

---

## 🛠️ Prerequisites

Before running any of these projects, ensure you have the following installed:
* Python 3.9+
* Pip (Python package manager)
* A **Groq API Key** (Get it from the [Groq Console](https://console.groq.com/))
